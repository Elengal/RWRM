# RimWorldRu — Полный русификатор RimWorld

Полный перевод RimWorld и всех официальных DLC на русский язык с набором дополнительных улучшений через Harmony-мод RuMod.

## Переводимые DLC

| DLC | Файлов перевода |
|-----|-----------------|
| Core | 722 |
| Ideology | 297 |
| Biotech | 224 |
| Odyssey | 202 |
| Anomaly | 180 |
| Royalty | 154 |
| DevMode | 36 |
| RimHUD (совместимость) | 7 |

**Итого: 1830+ файлов перевода.** Перевод нужных DLC подхватывается автоматически — достаточно, чтобы DLC был активирован в игре.

## Возможности RuMod

RuMod — это встроенный C#-мод (Harmony), который идёт вместе с переводом и расширяет игру:

### Перевод

- **Русские имена и фамилии** — полная замена NameBank: имена, фамилии, клички, родственники загружаются из файлов мода. Фамилии склоняются по полу, английские клички заменяются русскими.
- **Склонение возраста** — патч для RimHUD: корректное отображение «год/года/лет» в зависимости от числа.
- **Фамилия новорождённого** — при рождении ребёнку назначается правильная русская фамилия.
- **Пылесос DevMode** — сбор непереведённых строк из Dev-меню в JSON-словарь (для тех, кто занимается переводом).

### Геймплей

- **Снятие лимита фракций** — убирает ограничение на 12 фракций при создании мира. Можно добавить сколько угодно фракций.

### Визуальные улучшения

- **Фон главного меню** — выбор кастомного фона для главного меню (несколько вариантов или случайный).
- **Прозрачность панелей** — настройка прозрачности для панели переводчика, панели DLC и окна Debug log (10%–50%).
- **Перетаскиваемое окно переводчика** — можно двигать панель перевода на главном экране, позиция сохраняется.
- **Визуальные правки Debug log** — полупрозрачный фон, подстройка ширины под русские подписи.
- **Всплывающие подсказки в Dev-меню** — при наведении на пункт показывается полный текст команды.

### Для разработчиков

- **Логирование источников имён** — запись в файл, откуда взято каждое имя/фамилия/кличка при создании пешки.
- **Совместимость с другими модами** — RuMod автоматически отключает конфликтующие визуальные патчи.

## Установка

### Через Steam Workshop

Самый простой способ — подписаться на мод в Steam Workshop. Перевод активируется автоматически.

### Вручную

1. Скачайте последнюю версию из [Releases](../../releases)
2. Распакуйте папку `RimWorldRu` в директорию модов RimWorld:
   - **Windows:** `%USERPROFILE%\AppData\LocalLow\Ludeon Studios\RimWorld by Ludeon Studios\Mods\`
   - **Linux:** `~/.config/unity3d/Ludeon Studios/RimWorld by Ludeon Studios/Mods/`
   - **macOS:** `~/Library/Application Support/Ludeon Studios/RimWorld by Ludeon Studios/Mods/`
3. Запустите RimWorld, в мод-меню активируйте **RimWorldRu**
4. Мод должен загружаться **перед** Core (порядок определяется автоматически)

## Настройки

Все настройки доступны в меню мода: **Настройки → RimWorld RU**.

| Настройка | По умолчанию | Описание |
|-----------|-------------|----------|
| Фон главного меню | Случайно | Выбор фона: выкл / конкретный / случайно |
| Патчи NameBank | Вкл | Русские имена, фамилии, клички. Отключите при конфликтах |
| Снять лимит фракций | Вкл | Без ограничений на число фракций при создании мира |
| Перетаскиваемое окно переводчика | Выкл | Позволяет двигать панель перевода |
| Пылесос DevMode | Выкл | Сбор непереведённых ключей в JSON |
| Подсказки Dev-меню | Вкл | Всплывающие подсказки при наведении |
| Лог источников имён | Выкл | Запись происхождения имён в файл |
| Прозрачность панелей | 25% | Для панели переводчика, DLC, Debug log |
| Визуальные правки Debug log | Вкл | Полупрозрачный фон, подстройка ширины |

Конфиг сохраняется в: `Config\RimWorldRu\Config.xml`

## Структура проекта

```
RimWorldRu/
  1.6/
    Core/           — перевод базовой игры
    Royalty/        — перевод Royalty DLC
    Ideology/       — перевод Ideology DLC
    Biotech/        — перевод Biotech DLC
    Anomaly/        — перевод Anomaly DLC
    Odyssey/        — перевод Odyssey DLC
    DevMod/         — перевод DevMode
    RimHUD/         — совместимость с RimHUD
    Assemblies/     — скомпилированный RuMod.dll
    Textures/       — текстуры (фоны меню и пр.)
  About/            — метаданные мода (About.xml, иконка)
  LoadFolders.xml   — автозагрузка по DLC

RuMod_Source/       — исходный код RuMod (C# / Harmony)
  Main.cs           — точка входа мода
  ModSettings.cs    — все настройки
  Patches/
    Names/          — система русских имён
    Game/           — игровые патчи (фракции, меню, возраст)
    Debug/          — улучшения DevMode
    UI/             — визуальные патчи интерфейса
    RimHUD/         — совместимость с RimHUD
  Utils/            — утилиты (логгирование, переводчик)
```
